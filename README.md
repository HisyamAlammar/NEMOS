# NEMOS — Decentralized UMKM Revenue-Based Financing

![NEMOS Banner](https://images.unsplash.com/photo-1551288049-bbbda536ad3a?auto=format&fit=crop&q=80&w=1200&h=400)

## 🚀 Overview

**NEMOS** (Next-Gen Micro-business Operating System) is a decentralized investment platform designed to bridge the gap between retail investors and underbanked UMKM (Micro, Small, and Medium Enterprises). 

By leveraging **Revenue-Based Financing (RBF)** and **Blockchain Verification**, NEMOS provides a transparent, low-friction, and high-impact ecosystem where businesses grow through fair profit-sharing rather than predatory debt.

---

## ✨ Key Features

### 1. AI-Powered Investor Onboarding
An intelligent profiling system that analyzes risk tolerance and automatically maps investors to tailored UMKM portfolios (Konservatif, Moderat, atau Agresif).

### 2. Smart Contract Repayment Tracker
Real-time transparency. Every repayment transaction is recorded on the blockchain, providing investors with an immutable audit trail of their returns and UMKM health.

### 3. AI Learn Hub
A strategic educational suite designed to turn novice investors into analytical partners. The curriculum focuses on RBF fundamentals, risk mitigation, and financial statement analysis.

### 4. Dual-Mode Marketplace (UMKM Arena)
A vibrant, two-sided marketplace.
- **Investor Side**: Browse verified, graded UMKM listings with deep financial projections.
- **UMKM Side**: Manage omzet reporting, view funding progress, and access AI-driven business insights.

### 5. Impact Dashboard
Visualize the real-world change your capital creates. Track job creation, community empowerment, and SDG contributions directly within your portfolio view.

## 🏗 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/HisyamAlammar/NEMOS.git
   cd NEMOS
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

## 📈 Business Value Map (BMC Support)

NEMOS addresses the **$165B financing gap** for Indonesian MSMEs. By moving from a "Fixed Interest" model to a "Revenue Share" model, we reduce bankruptcy risk for business owners while providing attractive, impact-backed yields for investors.
